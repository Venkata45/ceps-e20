# 🎓 CEPS - College Event Processing System
### Final Project Report & Feature Roadmap

## 🚀 Vision
To provide a decentralized, transparent, and highly immersive ecosystem for managing university-wide events, bridging the gap between administrators, faculty, and students through real-time technology.

## 🏆 Core Feature Achievements (Implemented)

### 1. Unified Command Center (Admin)
- **Live Event Orchestration**: Seamless scheduling and management of FDP, SDP, CRT, and Cultural events.
- **Visual Analytics Hub**: Real-time data visualization using Recharts to track event distribution and registration trends.
- **Audit-Ready Export**: One-click CSV export for all event data, registrations, and attendance logs.
- **Smart User Management**: Centralized control over Students and Faculty with account locking/unlocking capabilities.

### 2. Digital Classroom Ecosystem (Faculty)
- **Paperless Attendance**: Mobile-responsive attendance marking with instant synchronization to the cloud.
- **Resource Repository**: Secure cloud-based uploading of event outcomes (PDF/PPT/Docs) directly accessible to students.
- **Outcome Tracking**: Transparent oversight of event impact and student engagement metrics.

### 3. Student Empowerment Portal
- **Immersive Discovery**: Beautifully designed event cards with real-time status indicators (Upcoming/Ongoing/Completed).
- **Secure payment Verification**: End-to-end UPI transaction tracking with automated email receipt generation upon admin verification.
- **Quality Feedback Loop**: Star-rating system for post-event evaluation to maintain institutional standards.
- **Personalized History**: Dedicated dashboard for registered events, certificates (receipts), and attendance records.

## 🛠️ Technical Excellence
- **Frontend**: React 18, Vite, Framer Motion (Animations), Tailwind CSS, Recharts.
- **Backend**: Node.js, Express, MongoDB Atlas, Nodemailer.
- **Real-Time**: Socket.IO for instant live updates across all dashboards.
- **Security**: JWT Authentication, Bcrypt password hashing, Role-Based Access Control (RBAC).

## 🌍 Deployment Strategy
- **Frontend**: Deployed on **Netlify** with SPA routing support.
- **Backend**: Hosted on **Render** (Node.js runtime).
- **Database**: **MongoDB Atlas** (Global Cloud Cluster).
